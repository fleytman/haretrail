# Examples

Sanitized examples and fixtures live here.

Status: Phase 3 migration started.

Examples must not contain:

- private source packets;
- real session debriefs;
- company-specific references;
- personal notes;
- local absolute paths.

Use small fictional examples that demonstrate the workflow shape.

## Available Fixtures

- `fixture-data-repo/`: a minimal fictional HARE Trail data repository for future smoke checks.

## Rules

- Keep examples small.
- Mark fictional content clearly.
- Prefer examples that test workflow shape over examples that look realistic.
- Do not copy real task folders, debriefs, notes, chat exports or company context into this directory.
